### 2021-11-19 refactoring

#### 1. ec2 서버 현재 시간 변경

- 시간이 글로벌 시간으로 설정이 되어 winston-daily-rotate-file 이 00:00 이 되어도 새로운 파일이 생성이 안됨
- 시간을 바꾸는 코드 2종류

```jsx
// 이쁜 창이 뜨면 Asia -> Seoul 로 설정 하면 됨.
sudo dpkg-reconfigure tzdata

// 원하는 시간대를 선택 하면 됨
sudo timedatectl set-timezone Asia/Seoul

// 시간 대 확인
date
// 또는
timedatectl

```
